sitsi: SOFT Inversion Toolkit for Synchrotron Images
====================================================

Welcome to the documentation for ``sitsi``, a Python toolkit which can be used
to calculate runaway distribution functions from
`SOFT2 <https://github.com/hoppe93/SOFT2>`_ Green's functions and experimental
synchrotron data.
